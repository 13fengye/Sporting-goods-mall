# Sporting-goods-mall
基于React和Django的体育用品商城网站

# 以下命令请用终端输入

# React


# Next.js and TypejScript
在项目文件夹Sporting-goods-mall下初始化项目：
npm init -y

添加React,Next依赖：
npm install --save react react-dom next

添加TypeScript开发环境依赖:
npm install --save-dev typescript @types/react @types/node

编译：
sudo npm run build

运行：
sudo npm run dev
